#说明
 该项目是采用Spring Security做的一个权限认证和授权的过程<br>
 * 根据给出的数据库脚本（tongauth.sql）建立数据库
 * 启动TongauthApplication.java
 * 输入http://localhost/login进行登录
 * 初始用户名和密码为admin和admin<br>
 
 需要注意
 * 角色名称必须以ROLE_开头<br>
